# Round Mountain Retreat Weddings Lookbook

Premium, cinematic 12-section web lookbook for Round Mountain Retreat weddings.

## Files
- `index.html` — 12 visual sections with floating nav, progress indicator, and cinematic layouts.
- `styles.css` — new premium design system, layouts, and motion.
- `review.html` — thumbnail overview with quick links into each section.

## How to view
Open `index.html` in a browser and scroll to navigate through the full-screen pages.
Open `review.html` for a visual table of contents.

## Notes
- Imagery uses Unsplash URLs as placeholders.
- Copy has been tightened for conversion-focused elegance.
- Added cinematic overlays, split layouts, collage sections, package cards, timeline strip, and testimonial cards.
- Improved mobile stacking and simplified nav patterns for smaller screens.
